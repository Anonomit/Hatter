1.1.0

Features

  • It's now possible to disable the default visibility settings. When disabled, new equipment will inherit visibility from the previously worn equipment in that slot.
  • Added Hatter WeakAura detection. The addon will check for the Hatter WeakAura on startup and will prompt the user to disable or delete it.

 

Interface Options

  • Changed defaults: Show Helm and Show Cloak. These options are now on by default.
  • New option: Force Default Visibility. When disabled, new equipment won't be forced as Shown or Hidden. This option is on by default.
  • New options: Forget Hats and Forget Cloaks. These buttons can be used to clear all items from memory.
  • New options: Disable WeakAura and Delete WeakAura. These buttons can be used to resolve the conflict with the Hatter WeakAura. They will only appear when the Hatter WeakAura is found.
