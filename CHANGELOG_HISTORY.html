2.0.0

Features

  • Large rewrite. Updated most addon code.
  • Added an options menu. Removed most slash commands, since they have been made redundant.
  • Items can now be ignored. Hatter doesn't change the equipment slot visibility when an ignored item is equipped.
  • Added an option to ignore all newly equipped items by default.

Changes

  • Removed WeakAuras detection. This was always kind of hacky, and the benefit is questionable due to the chance of false positives.

Fixes

  • Updated for WoW API changes.

Changes

1.1.0

Features

  • It's now possible to disable the default visibility settings. When disabled, new equipment will inherit visibility from the previously worn equipment in that slot.
  • Added an option to forget items, removing them from Hatter's memory.
  • Added Hatter WeakAura detection. The addon will check for the Hatter WeakAura on startup and will prompt the user to disable or delete it.

Changes

  • Changed defaults: Show Helm and Show Cloak. These options are now on by default.
